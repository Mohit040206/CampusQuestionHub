About Campus Question Hub

Campus Question Hub is a role-based academic paper sharing platform built as a real-world full-stack web application. The aim is to simplify access to previous year question papers for students and empower admins with a secure upload and management interface. This project is conceptualized, developed, and designed solely by Mohit Kumar Das as part of a learning initiative and a career-defining portfolio project.

Technologies Used

  • Backend: Java 21 with Spring Boot 3.3
  • Security: Spring Security 6.3 with custom login/session handling
  • Frontend: HTML5 + CSS3 (manually created without frameworks or JavaScript)
  • Database: MySQL
  • Build Tool: Maven
  • Version Control: Git + GitHub

Vision & Future Scope

This platform is not just an academic submission — it's a vision for the future. The ultimate goal is to scale Campus Question Hub for multiple colleges and universities, where each institution registers securely. Admins can upload materials tied to their institution code, and students can access papers only after student ID validation. Eventually, features like cloud-based file storage, OTP verification, and a page-reading AI module are planned to replace traditional systems.
